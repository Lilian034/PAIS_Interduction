TECH STACK

核心技術

🎤

ASR 自動語音辨識

將連續聲波轉換為文字，利用梅爾頻譜與深度神經網路學習語音與文字間的對齊關係

📝

NLP 自然語言處理

結合語言學與演算法，透過TF-IDF、Word2Vec或BERT等方式生成詞向量，理解語意與上下文

🔊

Speaker Encoder / TTS

擷取講者音色特徵，實現語音克隆。Text Encoder與Acoustic Decoder結合產生高品質語音

🤖

AI Agent / ML 機器學習

讓電腦從資料中學習規則並自行判斷，透過大量資料訓練模型掌握判斷模式

🔗

LangChain

為LLM打造的開源框架，串接提示模板、記憶、工具與檢索，讓單純問答升級為能規劃並執行任務的系統

📚

RAG 檢索增強生成

將多來源文本向量化並快速檢索，結合知識庫與AI模型產生準確且可追溯的回答
