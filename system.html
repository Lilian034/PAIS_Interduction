TECH STACK

核心技术

🎤

ASR 自动语音辨识

将连续声波转换为文字，利用梅爾频谱与深度神经网络学习语音与文字间的对齐关系

📝

NLP 自然语言处理

结合语言学与算法，透过 TF-IDF、Word2Vec 或 BERT 等方式生成词向量，理解语义与上下文

🔊

Speaker Encoder / TTS

提取讲者音色特征，实现语音克隆。Text Encoder 与 Acoustic Decoder 结合产生高品质语音

🤖

AI Agent / ML 机器学习

让电脑从资料中学习规则并自行判断，透过大量资料训练模型掌握判断模式

🔗

LangChain

为 LLM 打造的开源框架，串接提示模板、记忆、工具与检索，让单纯问答升级为能规划并执行任务的系统

📚

RAG 检索增强生成

将多来源文本向量化并快速检索，结合知识库与 AI 模型产生准确且可追溯的回答
